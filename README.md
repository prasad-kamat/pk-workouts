# Workout App — workout.pkamat.com

## Structure
```
├── backend/    → Node.js + Express + TypeScript + SQLite
└── frontend/   → Vite + React + TypeScript + Tailwind (PWA)  [Part 2]
```

## Backend Quick Start
```bash
cd backend
cp .env.example .env
npm install
npm run seed
npm run verify-api
npm run dev
```
